Play the Game

You are joining the best dice game in the world! Win all losers' bets and big dice win! The result of the dice roll is bounded to 1-6. If the dice results is bigger than others players, you win!

Create a Game

1. If you haven't already, download and install Metamask.
2. Enable the extension and sign in to your Metamask account.
3. Using the Chrome browser visit EthDice.

Join an Existing Game

1. If you haven't already, download and install Metamask.
2. Enable the extension and sign in to your Metamask account.
3. Using the Chrome browser visit EthDice.

Note: In the case of a win a 0% commission is subtracted from the total amount returned to the player. You will incur a small gas charge to place the bet. EthDice does not receive gas. Gas is used to maintain the Ethereum blockchain.
