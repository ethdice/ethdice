How to play

Rules

Big wins! The point value of a dice roll can range from 1 to 6. If your point value is bigger than other players', you win!

Create a Game

1. Create a game for 2 to 10 players (Tips: the less players the better, the more players the harder to end a game)
2. Set the Bet Amount for each player, ETH only
3. Pay your bet
4. Invite other players to place bets and join
5. The game starts automatically once all the players are in. The player of the largest dice point value wins the bets of all other players. If there are multiple winners, the losers' bets will be divided equally by all winners. If the game ends in a DRAW, all bets will be returned to the account of each player.

Join an Existing Game

1. Join a created game
2. Tap 'ROLL' to pay your bet, ETH only
3. Invite other players to place bets and join
4. The game starts automatically once all the players are in. The player of the largest dice point value wins the bets of all other players. If there are multiple winners, the losers' bets will be divided equally by all winners. If the game ends in a DRAW, all bets will be returned to the account of each player

Note: In the case of a WIN, no commission is needed for the first month. There will incur a small gas charge to place the bet. EthDice does not receive or take gas. Gas will be used to maintain the Ethereum blockchain.

Play Using Dapp Browser

1. Download and install Dapp Browser
2. Use Dapp Browser to visit EthDice.

Play Using MetaMask

1. Download and install Metamask
2. Enable the extension and sign in with your Metamask account
3. Use Chrome browser to visit EthDice.
